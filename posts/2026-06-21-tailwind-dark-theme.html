使用 Tailwind CSS 构建暗色主题

2026-06-21

为什么选择暗色主题？

暗色主题不仅护眼，还能让界面看起来更有质感。Tailwind CSS 提供了完善的暗色模式支持。

颜色系统设计

一个好的暗色主题需要一套协调的颜色系统：

用途 色值 说明
背景 #0f1117 主背景色
表面 #1a1b26 卡片、侧边栏
边框 #2f3146 分隔线、边框
文字 #c0caf5 主要文字
次要 #565f89 辅助信息
强调 #7aa2f7 链接、高亮

实际应用

/* tailwind.config.js */
colors: {
  dark: {
    bg: "#0f1117",
    surface: "#1a1b26",
    border: "#2f3146",
    text: "#c0caf5",
    muted: "#565f89",
    accent: "#7aa2f7",
  }
}

这套颜色系统灵感来自 Tokyo Night 主题，非常适合代码和技术类网站。
