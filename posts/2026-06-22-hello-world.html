为什么要写博客？

写博客是沉淀知识最好的方式之一。把学到的东西用自己的话写下来，比看十遍教程都管用。

这个博客用了什么技术？

  • 纯静态 HTML — 没有 React、Vue，就是最朴素的 HTML
  • Tailwind CSS — 实用优先的 CSS 框架
  • Markdown — 文章用 Markdown 写，构建时转成 HTML
  • GitHub Pages — 免费托管，零运维
